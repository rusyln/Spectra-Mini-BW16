## Hardware Components
- **BW16 Module** with RTL8720DN (Dual-Band Wi-Fi)
- **0.96** 128x64 OLED Display Module
- **3 Tactile Buttons** for user interface

## Features
- Dual-band 2.4GHz/5GHz Wi-Fi deauthentication
- TARGETED (PER CLIENT)
- RANDOM SSID
- BEACON + DEAUTH
- Interactive TFT display interface
- 3-button navigation control

## Pinout Configuration

### 0.96 128x64 OLED Display
| TFT Pin | BW16 Pin | 
|---------|----------|
| VCC     | 3.3V     |
| GND     | GND      |
| SCL     | PA25    |    
| SDA     | PA26    |      




### Button Connections
| Button  | BW16 Pin |
|---------|----------|
| UP    | PA13    | 
|  OK SELECT   | PA30    | 
| DOWN  | PA14      |
